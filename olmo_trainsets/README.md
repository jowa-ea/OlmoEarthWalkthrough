# OLMO training sets — DS3 & DS4 (workshop, repo-only)

Generated by `prepare_ds3_ds4_for_olmo.ipynb` (or the equivalent `main.py`)
in this folder's parent directory. Both datasets carry an
`original_class` column with the source string label, `time` =
`{year}-04-15`, and are split into `_partNNN` files of up to 100,000
rows/features each (OLMO's per-file limit).

These are the **repo-only** counterparts of DS3/DS4 from the broader
Ukraine winter-crop mapping pipeline. The production versions used
operationally there additionally blend in 2026 annotations not covered
by these two public repos.

---

## DS4 — Cropland status (`ds4_cropland_full_21-24`)

Source: `Ukraine_CIS_annotations_2021-2024` (shapefile), seasons 2021–2024
Date: April 15 of the annotation year

| class_cropland_full | label_cropland_full | original_class | Description |
|----------------------|----------------------|-----------------|-------------|
| 1 | Cropland | Cultivated | Active cropland |
| 2 | Non-cropland | Non-crop | Non-agricultural land |
| 3 | Abandoned | Abandoned | Abandoned cropland |
| 4 | Fallow | Fallow | Fallow cropland (normal rotation) |

Columns: `longitude, latitude, time, year, original_class, class_cropland_full, label_cropland_full`
Record count: 8000 (one row per point per annotated year, 2021–2024)

## DS3 — Crop type (`ds3_crop_types_22-25`)

Source: `Uk_sample_units_22-25` (GeoPackages), seasons 2022–2025
Date: April 15 of the annotation year
Note: only confirmed cropland with a known type is kept — `Non-crop`,
`Fallow`, and `Not_winter_cro` (2024–2025 catch-all) are dropped. Summer
crop was only annotated in 2022–2023.

| class_crop_type | label_crop_type | original_class | Description |
|-------------------|-------------------|------------------|-------------|
| 1 | Winter cereals | Winter_cer | Winter cereals |
| 2 | Rapeseed | Rapeseed | Rapeseed / winter oilseed |
| 3 | Summer crop | Summer_cro | Summer crops (2022–2023 only) |

Columns: `longitude, latitude, time, year, fid, original_class, class_crop_type, label_crop_type`
Record count: 2256 (GPKGs 2022–2025 combined, cropland-with-known-type only)
